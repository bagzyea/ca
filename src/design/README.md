# Design System Files

These are the original design exploration and wireframe components for Constitutional Africa.

## Files Overview

- **design-canvas.jsx** — Custom design system component framework
- **wf-primitives.jsx** — Wireframe UI primitives (boxes, buttons, labels, etc.)
- **brand-explorations.jsx** — Logo and color palette explorations
- **page-wireframes.jsx** — Page layout wireframes
- **app.jsx** — Main design system viewer application
- **hifi-*.jsx** — High-fidelity component designs
- **home-*.jsx** — Homepage layout variations
- **prebuild-*.jsx** — Pre-build stage components

These files were originally designed to run in a DesignCanvas viewer. As the application is developed, components can be extracted and integrated into the main app.
