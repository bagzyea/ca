# Constitutional Africa

A serious African intellectual platform for policy analysis, strategic research, and continental thought leadership.

## About

Constitutional Africa operates as a continental policy institute through:
- **Podcasts** — featuring leading African intellectuals and policy makers
- **Strategic Analysis** — deep dives into policy, economics, and constitutional matters
- **African Constitutional Economics (ACE)** — an emerging framework for African economic thought
- **Digital-First Content** — optimized for mobile access across Africa

## Core Values

- **Seriousness** — Continental policy institute standards
- **Identity** — Modern African, pan-continental perspective
- **Depth** — Intellectual rigor and strategic analysis
- **Access** — Mobile-first, digital-optimized platform

## Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Starts the development server at `http://localhost:5173`

### Build

```bash
npm run build
```

Creates optimized production build in `dist/` directory

### Preview Production Build

```bash
npm run preview
```
